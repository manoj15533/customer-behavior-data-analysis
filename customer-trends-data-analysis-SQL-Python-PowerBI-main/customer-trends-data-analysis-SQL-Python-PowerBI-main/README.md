# Customer Behavior Data Analysis

## Project Overview
This project analyzes customer shopping behavior to uncover patterns in purchasing,
discount usage, product preferences, and customer demographics using SQL, Python,
and Power BI.

## Tools & Technologies
- SQL (PostgreSQL)
- Python (Pandas, Matplotlib)
- Power BI
- Excel / CSV datasets

## Key Analysis Performed
- Gender-wise revenue analysis
- Impact of discounts on customer spending
- Top products by average review rating
- Shipping type vs purchase behavior
- Customer segmentation based on purchase patterns

## Files in this Repository
- `customer_shopping_behavior.csv` – Dataset
- `customer_behavior_sql_queries.sql` – SQL analysis queries
- `customer_behavior_dashboard.pbix` – Power BI dashboard
- `Business Problem Document.pdf` – Business understanding

## Learning Outcome
This project helped me strengthen my skills in SQL querying, data cleaning,
exploratory data analysis, and business-focused data storytelling.
